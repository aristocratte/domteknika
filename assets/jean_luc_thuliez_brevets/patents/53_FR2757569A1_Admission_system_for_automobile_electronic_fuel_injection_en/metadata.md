## Abrégé

The admission system has a housing containing a moving element (12) for regulating the airflow into the engine. This element (12) has a revolution symmetry around an axis of rotation. The housing (11) has a cavity (13) in which the regulating part (12) is located, so that it turns on the rotation axis fixed in the housing. The part (12) has a passage sensibly perpendicular to the axis of rotation and the housing has an admission opening and an escape opening. These openings arranged: (a) to be put into direct communication via the passage in the element (12) when the latter (12) is in a first `open' position and (b) to be closed by the element (12) when the latter (12) is in a second `closed' position and (c) to be put in regulated communication by the passage, when the element (12) is in a certain number of positions intermediate between the first and second positions.
