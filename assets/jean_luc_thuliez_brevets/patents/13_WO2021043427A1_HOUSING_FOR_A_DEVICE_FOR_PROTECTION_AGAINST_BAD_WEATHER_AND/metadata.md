## Abrégé

The present document relates to a device for protection against bad weather and/or sun, comprising: - a framework (12) that is extensible between a first closed framework (12) position and a second open framework (12) position, said framework (12) supporting a protective fabric (14); - a longitudinal axis mast (L) on which a sliding element (32) is to be mounted which, by virtue of sliding along said mast (18), is designed to make the framework (12) extend; and - a handle element (64) supported by one end of the mast (18); characterised in that it comprises a case in which all or part of the extensible framework (12) can be housed when in its first closed position, and in that the case (66) and the handle element (64) are configured to cooperate sealingly one with the other when the framework (12) is in its first position in order to form a sealed housing (62).
